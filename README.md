# OULAD Week-4 Early Outcome Prediction

This repository contains the individual project submission package for the OULAD week-4 early outcome prediction project.

## Submitted Package

All submitted material is kept under `Individual Project Submission/`:

- `How to Run_OULAD Individual Project (2550723)-1.pdf`: run instructions for the submitted project
- `OULAD_Individual_2550723-4.pdf`: submitted written report
- `OULAD_Individual_2550723-4.ipynb`: submitted notebook
- `submission_outputs-1.py`: submission output helpers
- `week4_pipeline-2.py`: submitted pipeline implementation

## Data

The raw OULAD CSV files are not included in this public repository. To rerun the workflow locally, place the dataset files in the project root as described in the submitted run instructions.
